## ✅ Task 1 — Exploring and Visualizing a Simple Dataset
### Objective:
Explore and visualize the Iris dataset to understand feature distributions and relationships between flower species.

### Skills Used:
- Data Exploration
- Data Visualization
- Exploratory Data Analysis (EDA)

### Dataset:
- Iris Dataset

### Key Insights:
- Analyzed sepal and petal measurements
- Visualized feature distributions using histograms and box plots
- Compared flower species using scatter plots and count plots

## ✅ Task 2 — Credit Risk Prediction
### Objective:
Predict whether a loan applicant is likely to default on a loan.

### Skills Used:
- Data Cleaning
- EDA & Visualization
- Classification Modeling
- Model Evaluation

### Models:
- Logistic Regression
- Decision Tree

### Key Results:
- Logistic Regression achieved ~88.6% accuracy
- Decision Tree achieved ~80.3% accuracy

## ✅ Task 3 — Customer Churn Prediction
### Objective:
Predict whether bank customers are likely to leave the bank.

### Skills Used:
- Feature Encoding
- Classification Modeling
- Feature Importance Analysis

### Models:
- Logistic Regression
- Random Forest

### Key Results:
- Random Forest achieved ~86.6% accuracy
- Logistic Regression achieved ~81.1% accuracy
- Important features included Age, Balance, and Credit Score
