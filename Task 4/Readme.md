📊 Insurance Claim Amount Prediction (Task 4)

📌 Objective  
The goal of this task is to predict medical insurance charges using a Linear Regression model based on personal and health-related information.

⚙️ Approach

- Performed data cleaning and preprocessing
- Checked for missing values
- Encoded categorical variables:
  - Label Encoding for sex and smoker
  - One-Hot Encoding for region
- Conducted Exploratory Data Analysis (EDA) using:
  - Histograms
  - Box plots
  - Scatter plots
- Visualized how:
  - BMI impacts insurance charges
  - Age impacts insurance charges
  - Smoking status impacts insurance charges
- Trained a Linear Regression model
- Evaluated model performance using:
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Squared Error)

📈 Results & Insights

- The Linear Regression model successfully predicted insurance charges.
- Model Performance:
  - MAE: ~4181
  - RMSE: ~5796
- Smoking status was one of the strongest factors affecting insurance charges.
- Higher BMI and increasing age were also associated with higher medical costs.
- The project demonstrated how regression models can be used for cost prediction and healthcare analytics.
