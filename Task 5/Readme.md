📊 Personal Loan Acceptance Prediction (Task 5)

📌 Objective  
The goal of this project is to predict which bank customers are likely to accept a personal loan offer using machine learning classification models.

⚙️ Approach

- Loaded and explored the Bank Marketing dataset
- Performed data cleaning and preprocessing
- Checked for missing values
- Encoded categorical variables using Label Encoding
- Conducted Exploratory Data Analysis (EDA) on:
  - Age
  - Job
  - Marital status
  - Loan acceptance patterns
- Trained two classification models:
  - Logistic Regression
  - Decision Tree
- Evaluated model performance using:
  - Accuracy
  - Confusion Matrix
  - MAE
  - RMSE

📈 Results & Insights

- Logistic Regression achieved the best performance with ~89.5% accuracy.
- Decision Tree achieved ~87.2% accuracy.
- Logistic Regression also produced lower MAE and RMSE values, indicating better prediction performance.
- Customer characteristics such as age, job type, and marital status influenced loan acceptance behavior.
- The project demonstrated how machine learning can help banks identify potential customers and improve marketing campaign strategies.
